# 项目需求文档

> 版本: 1.0 | 更新日期: 2026-07-14

---

## 项目背景

电动自行车（两轮电动车）已成为中国主流交通工具。然而，大多数厂商的电量显示采用纯电压查表法，忽略电池内阻和负载变化，导致SOC（State of Charge）显示严重不准——明明4格电骑了一周还是4格，但随时可能断电。

本项目的核心洞察：**同样油门开度下，不同电量时电机转速明显不同**。电机转速-电流-电池SOC之间存在可建模的耦合关系，利用这种关系可以显著改善SOC估算精度。

## 应用场景

- 模拟电动自行车整车电气系统（电机 + 电池 + 仪表）
- 台架演示：BLDC电机负载运行 + 电池放电过程 + SOC实时显示
- 面试展示：完整的感知→控制→执行→显示闭环

## 硬件平台

| 项目 | 选型 | 选型理由 |
|------|------|---------|
| MCU | STM32G431RB (NUCLEO-G431RB) | Cortex-M4F + HRTIM + 高速ADC + 内置运放/比较器，ST官方电机控制推荐 |
| 电机 | 2204/2205 BLDC云台电机 (KV1400) | 低压小功率，安全易调试，三相星型连接 |
| 驱动 | DRV8313 三半桥驱动 | 集成三相驱动+过流保护，SPI/PWM控制 |
| 电流传感 | INA226 (I2C) | 数字输出免模拟调理，16位分辨率，同步测电压+电流 |
| 显示 | SSD1306 OLED 128×64 I2C | P1已有驱动可复用 |
| 电池 | 3S 11.1V 2200mAh LiPo | 模拟电瓶车48V电池组的缩比版本 |

## 功能需求

### FR-1: BLDC电机基本控制
- **FR-1.1**: 6步换相开环控制（基于Hall传感器或反电动势过零检测）
- **FR-1.2**: 支持PWM占空比调节转速
- **FR-1.3**: FOC磁场定向控制（Phase 1进阶目标，可先用SimpleFOC库）
- **FR-1.4**: 速度闭环PI控制（参考来源：Hall频率或反电动势频率）
- **FR-1.5**: 启停控制 + 软启动（避免电流冲击）

### FR-2: 电机状态监测
- **FR-2.1**: 实时转速测量 (RPM)
- **FR-2.2**: 母线电流采样 (INA226)
- **FR-2.3**: 计算输入功率 (V_bus × I_bus)

### FR-3: 电池监测
- **FR-3.1**: INA226 读取电池电压 (精度 ±0.1% FSR)
- **FR-3.2**: INA226 读取母线电流 (采样电阻压降)
- **FR-3.3**: 可选NTC温度采集（ADC通道或DS18B20）

### FR-4: SOC估算
- **FR-4.1**: 库仑计数（电流时间积分）— 主要方法
- **FR-4.2**: 开路电压法（静止时OCV校正SOC）— 校正方法
- **FR-4.3**: 电机参数补偿（利用转速-电流-SOC耦合关系修正）— **创新点**
- **FR-4.4**: SOC百分比输出 (0-100%)

### FR-5: OLED显示
- 页0: 实时运行数据（SOC% / 转速 / 电流 / 电压）
- 页1: 系统状态（运行时间 / MCU温度 / 错误计数）
- 页2: 电池详情（累计Ah / 瞬时功率 / 电压跌落）
- 3秒自动翻页 + 按键手动翻页

### FR-6: 串口数据上报
- **FR-6.1**: 周期性上报（100ms间隔，可配置）
- **FR-6.2**: 数据格式: SOC / 转速 / 电流 / 电压 / 功率 / 温度
- **FR-6.3**: 支持ASCII文本格式（方便上位机/Python读取）
- **FR-6.4**: 支持命令控制：启停电机 / 设置转速 / 查询状态

### FR-7: 系统可靠性
- IWDG独立看门狗 (~1s超时)
- 电机过流保护（软件阈值 + 硬件DRV8313保护）
- 电池欠压保护（3S放电截止9V）
- 故障异常处理（HardFault/BusFault/UsageFault 输出诊断信息）
- 任务栈水位监控

## 非功能需求

### NFR-1: 实时性
- FOC电流环更新频率 ≥ 10kHz
- 速度环更新频率 ≥ 1kHz
- SOC更新频率 ≥ 10Hz
- OLED刷新频率 ≥ 3Hz
- 串口上报频率 ≥ 10Hz

### NFR-2: 安全性
- 过流保护响应 < 1ms
- 欠压保护响应 < 100ms
- 电机启动时软启动（缓慢增加占空比）
- 调试期间使用限流电源（建议设置3A限流）

### NFR-3: 可演示性
- 上电即可运行，无需PC连接
- OLED实时显示所有关键数据
- 演示视频可完整展示：电机运转 + 电池放电 + SOC变化
- 串口数据log可用于生成图表（Python matplotlib）

### NFR-4: 可维护性
- 模块化设计，职责清晰
- Doxygen标准注释
- 关键设计决策有文档记录
