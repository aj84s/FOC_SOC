# CLAUDE.md — 项目2 工作指引

## 项目概述

**项目名称**: 基于FOC的电动自行车智能电池管理系统

**一句话描述**: 用FOC控制BLDC电机（模拟电动车轮毂电机），同时采集电池电压/电流做SOC估计，利用电机运行参数补偿传统SOC算法的误差。

**MCU平台**: STM32G431RB (NUCLEO-G431RB, Cortex-M4F, 170MHz)
**RTOS**: FreeRTOS (CMSIS-RTOS v2)
**预算**: ~400元
**时间线**: 2026年7月中旬 → 8月底（约6周，9月秋招前完成）

## 工作原则

1. **增量推进**: 每次只改一个模块，改完验证通过再进入下一步
2. **可回退**: 每步改动都保证当前代码能编译、功能完整
3. **先文档后代码**: 修改前先确认对应的规范文档；新增模块按 `design_standards.md` 模板写注释
4. **Phase 门禁**: 每完成一个 Phase 必须通过全部验证清单项（见 `docs/dev_roadmap.md`）
5. **日志记录**: 每天开发结束时更新 `dev_log/YYYY-MM-DD.md`
6. **安全第一**: 电机控制代码必须包含过流保护；调试时使用限流电源
7. **先跑通再深耕**: 优先用成熟库（如SimpleFOC）跑通功能，再逐步替换为自己的实现
